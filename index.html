Browser Compatibility in a Nutshell


Browser Compatibility is the manner in which a web page looks in different web browsers. Different browsers read the website code differently. In other words, Chrome will render a website differently than FireFox or Internet Explorer will. It is important to ensure that your website is compatible across different browsers because not all people use the same browser. The major browsers for which a webdeveloper should ensure that a site is compatible include IE, Chrome, Firefox, Opera, Safari and on mobile devices.
